바이브 코딩이란?

AI와 협업하여 코딩 없이도 아이디어를 바로 웹앱으로 구현하는 직관적 개발 방식

Claude → GitHub → 웹페이지 생성 과정

1

아이디어 구상

수업에 필요한 도구나 활동을 구체적으로 설명합니다. "~한 기능이 있는 웹페이지를 만들고 싶어"

2

Claude와 대화

Claude에게 요구사항을 자세히 설명하고, 생성된 HTML 코드를 복사합니다.

3

GitHub 업로드

GitHub에 새 저장소를 만들고, index.html 파일로 코드를 업로드합니다.

4

웹페이지 발행

Settings → Pages → main → root → Save 순서로 GitHub Pages를 활성화하면 즉시 웹페이지가 생성됩니다. 학생들과 공유 가능!
